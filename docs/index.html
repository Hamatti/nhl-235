235

A CLI tool for NHL fans

Inspired by the visual aesthetics of YLE Teksti-tv's classic 235 page, I built a command line tool to give you the same feeling.

Install:

cargo install nhl-235
ln -s ~/.cargo/bin/nhl-235 /usr/local/bin/235
